# Whatsapp Web Clone
    A React based whatsapp web clone.
## Getting Started
![screenshot](https://user-images.githubusercontent.com/61149316/103240201-ad27cc80-4975-11eb-8df2-bef08ba58e43.JPG)

### Dependencies
* ReactJS.
* Material UI
### Installing
* Run the following command
```
npm run install
```
### Executing program
```
npm run start
```
